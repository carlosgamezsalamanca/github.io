Ana Resuelve Colmédica

Sitio de validación para analizar cómo un agente conversacional puede integrarse en una experiencia web corporativa con una presentación clara, institucional y alineada con una base documental definida.

Este sitio está siendo utilizado para realizar una prueba de concepto de las capacidades de Agentforce, en un entorno controlado que busca simular una experiencia digital corporativa alineada con el estilo visual de Colmédica. El propósito principal de esta validación es evaluar cómo un agente conversacional puede responder consultas, orientar al usuario y entregar información relevante de forma clara, estructurada y consistente, utilizando como soporte un conjunto específico de documentos guía previamente definidos como fuente principal de conocimiento.

Contexto de la prueba

La PoC busca validar no solo el funcionamiento técnico del agente embebido, sino también su comportamiento dentro de un canal web con apariencia institucional. En este ejercicio se analiza la calidad de la interacción, la capacidad del agente para comprender preguntas formuladas en lenguaje natural, la pertinencia de las respuestas generadas y su alineación con el tono esperado para una organización del sector salud. Asimismo, se pretende revisar la facilidad de adopción del canal, la consistencia en el uso del conocimiento disponible y el potencial de este tipo de solución como mecanismo de orientación y acompañamiento digital.

Objetivo funcional

El objetivo funcional de esta implementación es demostrar que Agentforce puede operar como un punto de contacto digital útil, confiable y escalable, integrándose en una página web con una identidad visual corporativa y ofreciendo una experiencia conversacional robusta. La prueba permite evaluar si el agente logra recuperar información pertinente, resumirla adecuadamente, responder con claridad y mantener un marco de conversación coherente frente a diferentes tipos de preguntas relacionadas con las guías cargadas.

Base documental utilizada

Para esta validación, el agente ha sido configurado para apoyarse en cuatro documentos específicos: la Guía Cardiología, la Guía de Gastroenterología PRE, la Guía de Vacunación y la Guía Dermatológica. Estas guías constituyen el núcleo de conocimiento de la prueba y permiten revisar cómo el agente interpreta contenidos especializados, identifica información relevante dentro del contexto disponible y la presenta de manera comprensible para el usuario final, manteniendo una comunicación organizada y útil.
